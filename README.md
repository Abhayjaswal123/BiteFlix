# 🍔 BiteFlix

A full-stack **MERN Food Discovery & Reels Platform** with authentication, category-based search, food reels, cart system, and an admin dashboard for managing content.

---

## 🚀 Live Demo

👉https://biteflix-frontend.onrender.com

---

## ✨ Features

### 🔐 Authentication System
- User Signup & Login (JWT Authentication)
- Secure protected routes
- Persistent user sessions

---

### 🍱 Food Feed Section
- Display all available food items in a modern feed UI
- Search food by name
- Filter food based on categories
- Smooth and responsive browsing experience

---

### 🎬 Reels Section
- Short food reels / video-style UI
- Scrollable feed experience similar to Instagram / TikTok
- Engaging visual food discovery

---

### 🛒 Cart System
- Add food items to cart
- Remove items from cart
- View selected items easily

---

### 🛠️ Admin Panel
Admin can:
- ➕ Add new food items
- ✏️ Update existing food items
- ❌ Delete food items
- Full control over platform content

---

## 🧠 Key Highlights
- Full-stack MERN application
- Role-based access (User & Admin)
- Category-based filtering system
- Reels-style UI for engagement
- REST API architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS / CSS3
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt.js

### Deployment
- Frontend: Vercel / Netlify  
- Backend: Render
